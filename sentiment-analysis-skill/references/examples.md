# 短评情感分析示例

## 正面示例

| 输入 | 情感 | 观点 |
|------|------|------|
| 这部电影太精彩了，强烈推荐！ | 正面 | 电影质量优秀，值得观看 |
| 老师讲得真好，通俗易懂 | 正面 | 教学质量高，易于理解 |
| 京东物流就是快，上午下单下午就到了 | 正面 | 物流速度快 |
| 这个手机性价比很高，买的值 | 正面 | 价格合理，值得购买 |
| 餐厅服务态度很好，菜品也很新鲜 | 正面 | 服务和品质都不错 |

## 负面示例

| 输入 | 情感 | 观点 |
|------|------|------|
| 质量太差了，不如不买 | 负面 | 质量令人失望 |
| 完全是浪费钱，坑爹 | 负面 | 不值这个价格 |
| 等了半小时还没上菜，体验极差 | 负面 | 等待时间过长 |
| 商家态度恶劣，完全不推荐 | 负面 | 商家服务差 |
| 好无聊，一点意思都没有 | 负面 | 内容乏味 |

## 中性示例

| 输入 | 情感 | 观点 |
|------|------|------|
| 还可以，普通水平 | 中性 | 质量一般 |
| 一般般，没啥特别的 | 中性 | 中规中矩 |
| 正常价格，正常品质 | 中性 | 符合预期 |
| 还行吧，说不上好坏 | 中性 | 无明显倾向 |
| 和之前买的差不多 | 中性 | 无差异化 |
