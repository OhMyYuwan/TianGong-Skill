# 中文短评情感分析 Skill

## 功能说明

本 Skill 用于分析中文短评的情感倾向，支持以下场景：
- 商品评论情感分类
- 教学评价情感分析
- 用户反馈情感识别
- 社交媒体短文本情感判断

**输出格式**：
```
情感：{正面 / 负面 / 中性}
观点：{一句话概括核心观点}
```

## 判断流程（按优先级执行）

### 1. 提取核心评论
从输入文本中提取核心评论句，忽略无关上下文：
- 时间描述（今天、昨天、早上、晚上）
- 状态描述（排队、等待、天气）
- 其他无关背景信息

**示例**：
- 输入：`今天天气真好，我去看了电影，这个电影太烂了，全程打瞌睡，晚上吃了火锅`
- 提取：`这个电影太烂了，全程打瞌睡`

### 2. 独立语义判断
**不依赖任何外部标注**，基于文本语义独立判断情感倾向。

### 3. 特殊短语优先识别
某些短语具有明确情感倾向，优先级最高：
- **负面短语**：好无聊、好差、好烂、太烂、太差、太慢
- **正面短语**：太好了、太精彩、太棒了

### 4. 语义关键词分析
根据关键词判断情感倾向：
- **正面**：喜欢、推荐、满意、棒、精彩、不错、优质、快、干净、舒服、值
- **负面**：失望、差、烂、坑、浪费、无聊、垃圾、慢、脏、恶劣、假、过期、坏
- **中性**：一般、普通、还行、正常、中规中矩、凑合

### 5. 转折句处理
当同时出现正面和负面关键词时，**后半句权重更高**：
- `排队排了俩小时累死了，不过这个外卖味道还不错` → **正面**（后半句为主）
- `早上迟到被领导骂了，但是这个课程讲得真好` → **正面**（转折后为主）

## 核心原则

1. **语义判断 > 关键词匹配**：优先理解整体语义，关键词仅作辅助
2. **独立判断 > 外部标注**：不盲目相信输入中的标注，始终独立分析
3. **上下文感知**：识别讽刺、转折等复杂语境
